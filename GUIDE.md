# 🪄 DMP Copilot 用户手册

本手册面向初次使用的同学，按步骤带你完成从安装到产出一份完整洞察报告的全过程。

---

## 1. 安装插件

DMP Copilot 是一个 Chrome 浏览器扩展，采用开发者模式手动加载。

**操作步骤：**

1. 获取插件源码文件夹 `DMP-Plugin`
2. 打开 Chrome 浏览器，地址栏输入 `chrome://extensions/` 并回车
3. 打开右上角的 **「开发者模式」** 开关
4. 点击左上角出现的 **「加载已解压的扩展程序」** 按钮
5. 选择 `DMP-Plugin` 文件夹，确认

加载成功后，扩展列表中会出现名为 **「Bzhi_DMP」** 的卡片，即为安装成功。

## 2. 首次启动

1. 打开达摩盘（dmp.taobao.com）任意人群透视页面
2. 页面右侧会出现一个竖排的 **「✨ 展开助手」** 按钮，点击它
3. 首次使用会弹出 **花名登记弹窗**，在输入框中填写你的花名或姓名（必填），然后点击「开启洞察之旅」
4. 花名只需填写一次，后续会自动记住

填写完花名后，你会看到引导面板上的 **4 步快速指引**，简单浏览即可了解核心流程。

## 3. 点亮雷达（获取数据凭证）

### 什么是雷达？

雷达是插件的"眼睛"——它通过拦截达摩盘底层请求，获取当前人群画像的数据访问凭证。**没有凭证，插件无法替你拉取数据**，所以每次打开达摩盘进行新分析前，都需要先点亮。

### 如何点亮（一般情况下是默认点亮）？

1. 在达摩盘页面左侧，随便点击任意一个画像标签（例如「用户性别」）
2. 触发页面加载画像图表
3. 观察插件面板左上角的 **心跳圆点**：
   - 🔴 **红色闪烁** = 尚未获取到凭证，无法提取
   - 🟢 **绿色常亮** = 雷达已锁定，可随时提取数据

### 4.1 普通标签

普通标签没有任何额外标记，**直接勾选复选框**即可加入提取队列。例如「用户性别」「用户年龄」「居住城市」等。

### 4.2 条件标签（带 ⚙️ 图标）

> ⚠️ **这是新手最容易踩坑的地方，请仔细阅读。**

条件标签的名称右侧会显示一个 **⚙️（齿轮图标）**，表示该标签需要你先在达摩盘页面上**手动配置下钻条件**，插件才能正确提取数据。

**操作流程：**

1. 在达摩盘页面左侧，找到并点击你想要的标签（例如「购物决策导向」）
2. 在弹出的配置面板中，设置你需要的下钻条件（比如选择具体的决策类型）
3. 点击页面上的「查询」或类似按钮，让图表加载出来
4. 此时插件会自动捕获你的下钻配置，标签图标从 **⚙️ → ✅(已就绪)**
5. 标签文字变为绿色，此时可以安全勾选

> ⚠️ 如果在图标仍然是 ⚙️ 的时候就勾选，插件会弹出警告提示，告诉你需要先配置下钻条件。

### 4.3 搜索过滤

标签面板顶部的搜索框支持**实时关键词过滤**。输入你想找的标签名（例如"消费"），面板会自动隐藏不匹配的标签，帮助你快速定位。**清空搜索框后，所有标签会恢复显示**，不用担心标签被搜"没"了。

### 4.4 清空勾选

标签面板右上角有一个 **「🧹 清空勾选」** 按钮。如果你勾选了大量标签后想重新选择，一键即可取消所有勾选，比手动逐个取消方便得多。

## 5. 一键提取

确认以下两件事就绪后，即可启动提取：

- ✅ 心跳灯为 **绿色常亮**
- ✅ 至少勾选了 **一个标签**

点击面板底部的 **「🚀 启动全维穿透提取」** 按钮，插件会逐个请求后台接口，几秒到十几秒后结果出现在右侧表格区域。

## 6. 看懂结果表格

### 各列含义

| 列名 | 说明 |
|------|------|
| 所属大类 | 标签的一级分类（用户特征/品类特征/渠道特征/私域特征） |
| 标签类型 | 标签的二级分类（如基础特征、消费特征等） |
| 标签名称 | 你在字典中选择的标签名 |
| 特征明细 | 该标签下的具体取值（如"男""女""18-24岁"） |
| 人群占比 | 该明细在当前人群中的原始占比 |
| Rebase | 智能归一化后的占比（详见下方说明） |
| CTR | 点击行为的 TGI 指数 |
| PPC | 转化行为的 TGI 指数 |

### 四大类颜色

| 大类 | 颜色 | 示意 |
|------|------|------|
| 用户特征 | 蓝色 | 个人基础属性 |
| 品类特征 | 橙色 | 商品/类目偏好 |
| 渠道特征 | 绿色 | 触达渠道偏好 |
| 私域特征 | 紫色 | 店铺资产相关 |

表格中「所属大类」和「标签类型」列的标签色块会根据上述配色自动区分，一眼就能分辨数据归属。

### 百分比进度条

「人群占比」和「Rebase」列带有**粉色/蓝色微进度条**，百分比越大进度条越长，方便横向比较不同明细的占比差异。

### Rebase 归一化是什么？

达摩盘底层有些标签是单选题（所有明细加起来约等于 100%），有些是多选题（加起来远超 100%）。Rebase 列会自动判断这种情况——单选题做归一化让比例更直观，多选题保持原值不变。**简单说：看 Rebase 列的数据，比看原始占比更"干净"。**

> - 绿色框标注「Rebase」进度条

## 7. 保存与复用方案

如果你有一套常用的标签组合（比如"每次分析都勾选用户性别 + 年龄 + 消费能力"），可以保存为方案，下次一键载入。

### 7.1 保存方案

1. 在标签面板中勾选你需要的所有标签
2. 在顶部搜索框中输入方案名称（如"我的标配模型"）
3. 点击 **「保存」** 按钮

保存后，方案会以胶囊形式出现在搜索框下方。

### 7.2 应用方案

点击方案胶囊会先展开操作按钮（应用 / 分享 / 删除），**再点击「应用」**才会自动勾选该方案中保存的所有标签。注意：点击胶囊本身只是展开菜单，不会直接应用方案，这是为了防止误操作。

### 7.3 分享口令

1. 点击方案胶囊，点击 **「分享」**
2. 口令（以 `DMP_MODEL_` 开头的一串字符）会自动复制到剪贴板
3. 将口令通过钉钉/微信/飞书发给同事即可

### 7.4 导入口令

1. 点击搜索框右侧的 **「导入」** 按钮
2. 粘贴同事发给你的口令
3. 确认后方案会自动出现在你的方案列表中

> ⚠️ 如果方案名称与已有方案重名，系统会询问是否覆盖。

### 7.5 删除方案

点击方案胶囊，点击 **「删除」** 并确认。

## 8. 导出数据

提取完成后，结果表格底部会出现两个操作按钮：

### 📋 复制至剪贴板

点击后，表格数据将以 **Tab 分隔格式** 复制到剪贴板。打开 Excel/飞书表格/Google Sheets，直接 `Ctrl+V` 粘贴即可，列会自动对齐。

### 💾 导出 CSV 文件

点击后自动下载一个 `.csv` 文件，文件名包含当天日期。文件使用 UTF-8 编码（带 BOM），直接用 Excel 打开不会乱码。

## 常见问题

### ❓ 为什么心跳灯一直是红色的？

可能的原因及排查步骤：

1. **你还没有在页面左侧点击任何画像标签。** 插件需要拦截一次达摩盘自身的画像查询请求才能获取凭证。在页面左侧随便点一个标签，等图表加载出来再回来看心跳灯。
2. **你在打开插件面板之后才点击的标签。** 如果你先点了标签再打开插件面板，插件可能没有捕获到。尝试切换一个标签重新点击。
3. **页面有 iframe 或弹窗遮挡。** 确保你在达摩盘的主操作窗口内点击标签，不要在弹出的小窗口中操作。
4. **如果以上都无效**，刷新页面重试，先点标签再打开插件面板。
